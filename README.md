Authentication-PassportJS
=========================

A simple authentication system (Node.js, MongoDB, Passport.js and Angular.js)

## Install
  - Node.js
  - npm
  - bower
  - MongoDB
  
## Edit /config/db.js
  - Type your MongoDB url where it's required
  
## NPM
 ```sh
$ npm install
```

## Bower
 ```sh
$ bower install
```

## Node.js
 ```sh
$ node server
```

Finally go on http://localhost:8080/
